Charging by Induction:

This application illustrates how to charge a conducting object by induction. The user can charge the sphere external to the square conductor by using the slider labeled "External Charge". The square conductor can be grounded by using the button so labeled, and ungrounded by using the corresponding button.

Play around with charging the External Charge and grounding and ungrounding the box to see what sequence will give you a net positive charge on the box with zero external charge. Do the same thing to see how you would end up with a net negative charge on the box.
